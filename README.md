# Portfolio Website

This is a personal portfolio website built with React and Vite. It showcases my skills, projects, education, and contact information.

## Features
- Responsive design
- Animated transitions
- Downloadable resume
- Sections: About, Skills, Education, Projects, Contact
- Modern UI

## Technologies Used
- React
- Vite
- JavaScript (ES6+)
- CSS

## Folder Structure

## Getting Started
1. **Install dependencies:**
   ```powershell
   npm install
   ```
2. **Run the development server:**
   ```powershell
   npm run dev
   ```
3. **Open in browser:**
   Visit `http://localhost:5173` (default Vite port)
